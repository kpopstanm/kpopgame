Sobre o site:

O Kpopgame é um site indie desenvolvido com o intuito de criar Mini Games divertidos relacionados ao mundo do Kpop.

Como jogar

Para jogar, apenas preencha o formulário abaixo com os dados que se pede*, logo após você sera redirecionado(a) para os minigames, e poderá escolher algum apenas clicando no nome.



*Por ser um site indie, mais especificamente feito por duas pessoas, não conseguimos e nem temos as ferramentas necessárias para desenvolver os minigames no site em si. Por isso, usamos uma ferramenta de programação (Html + Javascript) Para enviar os links dos mini games por email. É tudo 100% automático e feito pelo próprio sistema, e seus dados são automaticamente apagados do servidor assim que os mini games forem enviados!
